# goneovim 設定ファイル

## ファイル構成

### ソフトウェア設定

* `setting.toml` - 設定ファイル

設定方法の詳細は

```vim
:help goneovim-configration
```

### デスクトップエントリ

* `goneovim.desktop` - デスクトップエントリ
* `goneovim.ico` - アイコン

### スクリプト

* `setup.sh` - ソフトウェア設定
* `unzip_goneovim.sh` - [github](https://github.com/akiyosi/goneovim/releases)で配布された圧縮ファイルからgoneovimを実行できるよう配置
